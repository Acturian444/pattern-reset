# Handoff documents (start here)

Use these when starting a **new AI chat** so work continues without re-explaining the project.

| Document | Purpose |
|----------|---------|
| **[HANDOFF-FULL-CONTEXT.md](./HANDOFF-FULL-CONTEXT.md)** | **Primary:** tech stack, env, routes, file map, Personal Relationship Read funnel, quiz/results touchpoints, UX decisions, troubleshooting, cache-bust versions. |
| **[HANDOFF-QUICK-REFERENCE.md](./HANDOFF-QUICK-REFERENCE.md)** | One-page checklist: key URLs, storage keys, commands, files to open first. |

**How to use:** In a new chat, say something like: *Read `docs/HANDOFF-FULL-CONTEXT.md` and `docs/HANDOFF-QUICK-REFERENCE.md` in this repo before making changes.*

Related strategy copy lives in `docs/mastermind/` (see `PLAYBOOK-INDEX.md`). Business playbooks live in `Alex Hormozi Mastermind/*.md`.
