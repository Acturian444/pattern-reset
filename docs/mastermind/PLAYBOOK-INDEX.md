## Route by question type (fast map)

Use this when you are stuck choosing **which** playbook to open first.

| You are trying to… | Start here |
|--------------------|------------|
| Headlines, angles, first lines, scroll-stopping ideas | Hooks |
| Creative, ads structure, creative testing | Goated Ads |
| Full acquisition system, channels, cadence | Marketing Machine |
| What to charge, guarantees, stacks, “too expensive” | Pricing → Price Raise |
| Raise prices without blowing up retention | Price Raise + Retention |
| Email/SMS sequences, follow-up after opt-in | Lead Nurture |
| Long-term revenue per customer, ascension | Lifetime Value + Retention |
| Objections, calls, “no,” blame, closing loops | Closing |
| Brand vs direct response, “cool factor,” associations | Branding |
| Short-term revenue hit, quick leverage | Fast Cash |

## Route by funnel stage

| Stage | Primary playbooks |
|-------|-------------------|
| Cold attention (ads, short video, hooks) | Hooks, Goated Ads |
| Landing / quiz framing | Hooks, Branding (if “why you” matters), Pricing (if paywall or bump) |
| After opt-in | Lead Nurture |
| Sales conversation / DMs / calls | Closing, Pricing |
| Churn / repeat / ascension | Retention, Lifetime Value |

## How to ask the AI (copy-paste template)

```text
Context: [1–2 sentences: offer or quiz, audience, constraint]
Goal: [one outcome: leads, CVR, price, LTV, etc.]
Use: @docs/mastermind/PLAYBOOK-INDEX.md and @[relevant playbook .md files]
Answer format: Situation (1 line) → Recommendation → Why → Risks → Best next action
Ground answers in the playbooks; if something isn’t in the docs, say what’s missing.
```
