# Landing Page Color System

## Section Backgrounds (Pattern Quiz Landing)

| Section | Background | Purpose |
|---------|------------|---------|
| **Hero** | `#050505` (near black) | Bold opening, high contrast |
| **Pain-clarity** | `linear-gradient(#0a0a0a → #141414 → #161616)` | Dark block continues from hero; gradient flows smoothly |
| **Quiz** | `#fffcf1` (brand cream) | Light, readable; matches brand |
| **FAQ** | `#fffcf1` | Consistent with Quiz + Footer |
| **Footer** | `#fffcf1` | Brand consistency |
| **Body / Header** | `#fffcf1` | Site-wide brand base |

## Brand Colors

- **Primary accent:** `#f10000` (red)
- **Light base:** `#fffcf1` (cream)
- **Dark base:** `#050505` → `#161616` (gradient family)

## Design Pattern

- **Dark block:** Hero + Pain-clarity (emotional hook)
- **Light block:** Quiz + FAQ + Footer (action + support)
- **Transition:** Subtle red gradient line (`rgba(241, 0, 0, 0.15)`) at top of Quiz section echoes Hero's bottom accent
